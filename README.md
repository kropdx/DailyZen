# zen

A mindfulness break for developers. Random zen quote + guided 4x4 breathing exercise, right in your terminal.

## Install

```
/plugin marketplace add kropdx/zen
/plugin install zen
```

## Usage

```
/zen
```

That's it. A random quote appears, followed by a 60-second animated breathing exercise (4 rounds x 4 phases x 4 seconds).

## What it does

1. Displays a random zen quote from a curated collection of 200+
2. Runs a 4x4 box breathing exercise with animated progress bars
3. Returns you to your work, refreshed
